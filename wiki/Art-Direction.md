# Art Direction

## Visual Style

Competitive multiplayer kitchen battle with stylized 3D characters.

## 3D Models

- Chef characters with animated models
- Station models (cooking, chopping, plating)
- Ingredient models
- Map environment assets

## Animations

DOTween-based animations for:
- UI transitions (USS-based, see [UI/UX](UI-UX.md))
- Character animations
- Station interactions
- Visual effects

## UI Art

- USS stylesheets for all UI elements
- SkewedBoxElement custom control for stylized panels
- Theme system with consistent color palette

## Technical Art

- Shader-based effects for fire, steam, etc.
- Particle systems for cooking effects
- Post-processing for match atmosphere
