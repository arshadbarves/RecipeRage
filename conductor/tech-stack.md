# Tech Stack

## Core Technologies
- **Game Engine:** Unity (Version 6000.3.0f1)
- **Programming Language:** C# (.NET 4.x or newer)
- **Development Environment:** Visual Studio 2022 or JetBrains Rider

## Backend & Services
- **Epic Online Services (EOS):** Used for Authentication, Friends system, Presence, and P2P communication.
- **Firebase:** Used for Remote Config, Analytics, and potentially crash reporting.
- **Monetization:** Integrated Rewarded Ads, Season Pass system, and In-game Shop functionality.

## UI & Architecture
- **UI Framework:** Unity UI Toolkit (UXML for templates, USS for styling).
- **Architecture:** Modular, extensible framework with clear separation of concerns (Core, Modules, UI).
- **State Management:** Custom state machine for game flow (Bootstrap, MainMenu, Gameplay).

## Version Control & DevOps
- **Git:** Primary version control system.
- **Git LFS:** Used for managing large binary assets (textures, models, audio).
